[Coffee] 微軟經理的演講

2005-10-29

雖然興趣不是很大，不過一個禮拜只有這堂課…所以還是出門了。

今天講的題目是 KM（知識管理），說實在話，我的印象是從講者的一句：「用 Outlook Express 代表你的 KM 程度很低」開始的。當講者問：「你們平常是用 Outlook 還是 Outlook Express」時，我直覺爆出的一句話是：從來沒用過。

後來看他又繼續介紹＂使用好用又方便的 Web 介面來管理伺服器，新增有權限控管的連線帳號＂，我第一個想到的又是 Webmin。

最後講到下一版 Windows 會有的搜尋功能，這個我就比較了解了，因為跟 Mac 的 Spotlight 以及 Windows 上的 Google Desktop Search 一樣，甚至 Linux 上也有類似的方案。

好吧，既然他上面說的所有需求我第一個想到的都不是微軟提供的方案，所以最後演講結束之後，我跑去問講者：「關於剛剛提到的搜尋功能，目前市面上除了 Windows 之外的所有作業系統都有了，Windows 現在才要開始做，還有什麼優勢呢？」對方回答：「我剛剛說過了啊，這個搜尋是要把非結構化的資料變成結構化的，這樣搜尋才快，也才能根據資料的內容做搜尋啊」標準的答非所問…，為了不要給他太難堪，所以我只好說：「喔，那就是跟 Apple 的 Spotlight 一樣嘍？那為什麼微軟現在才開始補這個洞呢？」講者又回答了：「呃…因為一個技術要有客戶需要我們才會去做，因為之前沒有這種需要，所以我們沒有做」。

聽完他說的，我心裡面只有一個感想：＂微軟會被人瞧不起不是沒有原因的＂。他們根本不會去幫顧客找出需求，一點創新能力都沒有，看到市場上有什麼東西反應不錯，就跟著加到自己的產品中。

對比微軟的市佔率，這叫小人得志嗎？
