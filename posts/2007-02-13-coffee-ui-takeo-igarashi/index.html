日本的 UI 大師 Takeo Igarashi

2007-02-13
Comments

今天在 HEMiDEMi 看到有篇書籤是關於 UI （使用者界面）的，點進去一看，文章內容是介紹日本的 Takeo Igarashi，文章中也介紹了他到目前為止的一些主要成就。

除了瞻仰大師風采之外，這篇文章還有一句話讓我心有戚戚焉，

越簡便的UI所意味的也就是背後越複雜的運算和演算法

使用者追求的當然是方便好用的 UI，但是相對的，程式設計師就要付出更多的心力在背後一些看不到的部份。這點，台灣的軟體公司作的實在是差勁，像我偶爾使用 PowerDVD 的時候，除了那個大大的播放按鈕和快轉、倒轉按鈕之外，其他的按鈕我實在是看不出來代表什麼意思，每次用一用就放棄了…我沒有要求這個軟體能猜到我心裡想什麼，但是好歹也要讓我能猜到要怎麼讓你工作吧！看來，台灣的軟體之路還很漫長啊…
